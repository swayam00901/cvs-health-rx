# AI-Assisted Development Workflow

This project is designed to be co-developed with AI coding assistants. Notes for new contributors:

- **Scaffolding:** Initial project structure generated by Claude + Cursor, then hand-reviewed. See commit history.
- **Code review:** PRs run through a Claude Code reviewer skill (`/review-pr`) before human review. Reviewer focuses on: React Native perf (re-renders, list virtualization), SQL N+1 in EF Core, missing `CancellationToken` plumbing, PHI leakage into logs.
- **Test generation:** Use Copilot Chat or Claude to generate Jest + xUnit test stubs from controller contracts; humans write the assertions.
- **FHIR mapping:** EHR DTOs are kept close to FHIR R4 so that AI-assisted mapping tools can ingest HL7 FHIR bundles with minimal hand-tuning.
- **Guardrails:** Never paste real patient data into any AI prompt. A pre-commit hook runs `detect-secrets` + a custom regex for MRN/SSN/DEA formats.
